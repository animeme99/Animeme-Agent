# Risk Review

Use before escalating any Animeme trend into a public thesis or watch plan.

Workflow:

1. Run `npm run risk -- --topic <topic-id>`.
2. Check token visibility, liquidity, summary quality, and board persistence.
3. Mark missing data as risk, not as neutral.
4. Recommend stopping if the topic drops out of Animeme public modes.

Rules:

- No wallet signing.
- No live trading.
- No claims of safety unless verified from public data.
